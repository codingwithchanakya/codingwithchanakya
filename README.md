## python basics

This is a __print function__ this display's the text inside of ` "" ` (or)` '' `.

```python 
print("Hello, World")
```

If you want to print more then one lines in python with a line of code you use ` \n ` as shown in the code below.

```python
print(" Hi,\n I'm Chanakya")
```
this is the most smallest way to print multiple line.

_2-examples_ for printing multiple lines :

1. multiple print functions :
```python
print("Hi,")
print("I'm chanakya")
```

2. multiple line string
```python
print("""Hi,
         I'm chanakya""")
```
